# Kebijakan Privasi

Terakhir diperbarui: 5 Mei 2026

Website ini dikelola oleh **SMAS PGRI 1 Bandung** (selanjutnya disebut sebagai "Kami"). Kebijakan Privasi ini menjelaskan bagaimana Kami melindungi privasi pengunjung website.

## 1. Dasar Hukum Perlindungan Data

Kami berkomitmen untuk menghormati dan melindungi privasi pengguna sesuai dengan ketentuan peraturan perundang-undangan yang berlaku di Republik Indonesia, termasuk Undang-Undang Nomor 27 Tahun 2022 tentang Pelindungan Data Pribadi (UU PDP).

Meskipun website ini tidak mengumpulkan data pribadi pengguna, Kami tetap menerapkan prinsip-prinsip perlindungan data seperti transparansi, keamanan, dan pembatasan penggunaan data.

## 2. Informasi yang Kami Kumpulkan

Kami tidak mengumpulkan data pribadi apa pun dari pengunjung website ini.

- Tidak ada proses pendaftaran atau login
- Tidak ada formulir pengisian data
- Tidak ada pengumpulan nama, email, atau informasi pribadi lainnya

## 3. Penggunaan Cookie dan Penyimpanan Lokal

Website ini tidak menggunakan cookie.

Kami hanya menggunakan **local storage (penyimpanan lokal pada browser)** untuk menyimpan preferensi tampilan, seperti:

- Mode tema (mode gelap / mode terang)

Data ini:

- Tidak bersifat pribadi
- Tidak dikirim ke server mana pun
- Hanya tersimpan di perangkat pengguna

## 4. Penggunaan Informasi

Karena Kami tidak mengumpulkan data pribadi, maka:

- Tidak ada data yang diproses, dianalisis, atau disimpan
- Tidak ada aktivitas pelacakan pengguna

## 5. Berbagi Informasi kepada Pihak Ketiga

Kami tidak membagikan data apa pun kepada pihak ketiga karena tidak ada data pribadi yang dikumpulkan.

## 6. Tautan ke Situs Lain

Website ini dapat berisi tautan ke situs eksternal sebagai referensi atau informasi tambahan.

Kami tidak bertanggung jawab atas konten maupun kebijakan privasi dari situs pihak ketiga tersebut.

## 7. Privasi Anak

Website ini ditujukan untuk umum, termasuk pelajar.

Kami tidak secara sengaja mengumpulkan data pribadi dari siapa pun, termasuk anak di bawah umur.

## 8. Keamanan

Karena tidak ada data pribadi yang dikumpulkan atau disimpan, risiko kebocoran data pribadi melalui website ini sangat minimal.

## 9. Perubahan Kebijakan Privasi

Kebijakan Privasi ini dapat diperbarui sewaktu-waktu.

Setiap perubahan akan ditampilkan pada halaman ini dengan tanggal pembaruan terbaru.

## 10. Persetujuan

Dengan menggunakan website ini, Anda dianggap telah membaca dan menyetujui Kebijakan Privasi ini.
